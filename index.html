Custom Master
Este é um Script para Dragon Block C que consiste em voce criar um NPC com um certo nome e ja vai funcionar automaticamente. O NPC funciona como um vendedor de Formas — ou seja, ele permite que jogadores comprem formas específicas oferecidas pelo servidor. A ideia é fazer algo diferente do Dragon Block C convencional, enquanto no DBC normal vc clica no "V" e pode upar a super form (liberando no max umas 7 ou 8 formas, com esse script voce pode colocar uma variedade absurda de formas como o SSJ5 que e algo que todos sempre queriam no jogo e nunca foi adicionado! Nesse script alem de poder adicionar a forma que quiser voce pode tambem alterar o preço cobrado por essa forma [ o valor e cobrado em TP do comprador]

Funcionalidades principais: Um NPC que vende forms por TP (training points)

Warps (comando .warps)
Este é um Script para Dragon Block C que adiciona o comando ".warps" ao seu jogo seja offline ou online. Com esse comando, jogadores podem ver uma lista de warps e teletransportar-se para diversos locais do mundo. A configuração é simples: você define os locais, nomes e o comando passa a oferecer essas opções no jogo.

O diferencial: tudo é totalmente configurável — nome do warp, coordenadas e possibilidade de criar warps rápidos (como um .kami que te leva direto para o kami) para áreas especiais do servidor.

Bank TP
Este é um Script que adiciona um sistema de banco de TP e uma Lojinha ao Dragon Block C. Com esse sistema, jogadores podem:
  • depositar TP;
  • sacar TP;
  • transferir TP entre Players;
  • aumentar o limite máximo de TP que um jogador pode ter armazenado no Banco.
Junto ao sistema de banco há uma loja integrada onde os jogadores podem comprar efeitos e itens: Marca Majin, Ki Divino, Ki Lendário, Senzu e Carne de Dinossauro, entre outros itens que você queira disponibilizar. A loja é ligada ao banco: compras descontam o TP ou outra moeda configurada.
